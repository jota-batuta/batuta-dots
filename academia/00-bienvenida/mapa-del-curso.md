# Mapa del curso

Aqui tienes todas las lecciones organizadas. Elige tu ruta segun tu perfil, o simplemente sigue el orden de arriba a abajo.

---

## Rutas recomendadas

### Ruta Estudiante
Para estudiantes de universidad, personas aprendiendo a programar, o curiosos sin experiencia tecnica.

```
Que es Batuta Dots → Antes de empezar → Tu primer proyecto →
El pipeline SDD → Skills que tienes → Casos: tesis / prototipo startup
```
**Tiempo estimado**: 4-6 horas

### Ruta Desarrollador
Para programadores que quieren maximizar su productividad con IA.

```
Tu primer proyecto → Anatomia de un comando → El pipeline SDD →
Skills que tienes → Agentes y equipos → Depurando tu trabajo →
Equipos de agentes → Casos: SaaS / API / Landing
```
**Tiempo estimado**: 6-8 horas

### Ruta Operaciones / Industria
Para profesionales de operaciones, mantenimiento, produccion, logistica, o cualquier area no-tecnica.

```
Que es Batuta Dots → Antes de empezar → Tu primer proyecto →
El pipeline SDD → Los gates → Casos: tu industria
```
**Tiempo estimado**: 3-5 horas

### Ruta CTO / Lider tecnico
Para quienes necesitan entender el ecosistema completo y extenderlo.

```
El pipeline SDD → Los gates → Skills que tienes → La capa CTO →
Depurando tu trabajo → Equipos de agentes → Hooks y automatizacion →
Extendiendo el ecosistema → Templates de equipo → Referencia completa
```
**Tiempo estimado**: 8-10 horas
