4b. Ciclo iterativo CTO ↔ Code via Notion

El flujo es iterativo. Discoveries en Code se escriben a Notion KB inmediatamente. CTO los lee y ajusta. Sin copy/paste, sin cambiar de ventana para transferir archivos.
